Quem Somos

Somos uma empresa de engenharia de software que desenvolve produtos e serviços com foco em inovação.

A Bit101 acredita no poder da tecnologia de software para potencializar, remodelar e otimizar a comunicação entre indivíduos.

Inovação

Inovação e criatividade fazem parte do nosso DNA e estamos sempre à procura de novidades, possibilidades e tecnologias para agregar valor aos projetos e oferecer serviços e produtos diferenciados.

Desenvolvimento

Integramos técnicas sólidas de desenvolvimento de software para criar soluções seguras, rápidas e sob medida.

O que fazemos

Dedicamos nosso tempo transformando ideias em ferramentas.

Desenvolvimento Web

Aplicativos, websites responsivos, portais institucionais, hotsites promocionais, peças de mídia online e sistemas web.

Desenvolvimento Móvel

Aplicativos nativos para Android, aplicativos híbridos (web) para iOS. Sistemas backend para aplicativos móveis.

Webservices e APIs

Análise, projeto e desenvolvimento de sistemas backend.

Design, Criação e Concepção

Layouts atraentes e interfaces intuitivas com foco na experiência e usabilidade do produto.

Consultoria de Software

Concepção, análise de requisitos, arquitetura, análise e projeto detalhado de software
